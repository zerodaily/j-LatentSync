# 本地部署注意事项

## 环境要求

### 硬件要求
- NVIDIA GPU with at least 12GB VRAM (recommended 24GB+)
- CPU: Modern multi-core processor
- RAM: 16GB minimum, 32GB recommended
- Storage: 20GB+ free space (SSD recommended)

### 软件要求
- Ubuntu 20.04+ / Windows 10+ / macOS 12+
- CUDA 11.8+ and cuDNN 8.x
- Python 3.10+
- Conda or virtual environment recommended

## 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/zerodaily/j-LatentSync.git
cd j-LatentSync
```

2. 创建并激活虚拟环境
```bash
conda create -n j-latentsync python=3.10
conda activate j-latentsync
```

3. 安装依赖
```bash
pip install -r requirements.txt
```

4. 下载预训练模型（参考官方文档）

## 常见问题

### CUDA版本不匹配
确保安装与你的GPU兼容的CUDA版本

### 显存不足
- 降低batch size
- 使用更小的模型
- 启用GPU优化选项

### 依赖安装失败
使用conda安装部分依赖后再pip install其他依赖
