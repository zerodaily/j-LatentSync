# 启动命令

## 基本启动

### WebUI模式
```bash
python app.py
```

### API服务器模式
```bash
python api.py
```

### 带参数的启动
```bash
# 指定端口
python app.py --port 7860

# 指定GPU设备
python app.py --device 0

# 启用混合精度
python app.py --precision fp16

# 完整示例
python app.py --port 7860 --device 0 --precision fp16 --batch_size 4
```

## Docker部署

```bash
# 构建镜像
docker build -t j-latentsync .

# 运行容器
docker run --gpus all -p 7860:7860 j-latentsync
```

## Docker Compose

```bash
docker-compose up -d
```

## 验证运行

启动后访问：
- WebUI: http://localhost:7860
- API: http://localhost:8000/docs
